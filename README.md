# Programación II - Trabajos Prácticos

Este repositorio contiene las resoluciones de los **Trabajos Prácticos** de la materia **Programación II (Programación Estructurada)** de la **Tecnicatura Universitaria en Programación (UTN)**.  

Cada TP está implementado en **Java**, utilizando únicamente estructuras básicas: condicionales, bucles, funciones, arrays y recursividad.

---

## 📂 Organización

- Cada **TP** se entrega en un único archivo Java (`TP1.java`, `TP2.java`, `TP3.java`, etc.).
- Dentro de cada archivo, los **ejercicios** están comentados y separados.  
- Para **probar un ejercicio**, simplemente se deben **descomentar las líneas** que corresponden a ese bloque de código.

Ejemplo:  
// !!!!!!!!!!!!!!!!!!   EJERCICIO 5   !!!!!!!!!!!!!!!!!!  
// /*  
// ... código del ejercicio ...  
// */  


👉 Para ejecutarlo, borrar /* y */.



⚠️ Importante: solo un ejercicio debe estar descomentado en el main, al momento de ejecutar el documento. Si hay mas de un archivo descomentado, generara errores.
